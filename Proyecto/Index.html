Habilidades y Conocimientos

HTML

HTML es el lenguaje que estructura las páginas web mediante etiquetas, definiendo elementos como texto, imágenes y enlaces.

CSS

CSS es un lenguaje usado para estilizar y dar formato a las páginas web, controlando colores, fuentes y disposición de los elementos.

JavaScript

JavaScript es un lenguaje de programación que agrega interactividad a las páginas web, permitiendo dinamismo y acciones en tiempo real.

Python

Python es un lenguaje versátil utilizado en desarrollo backend y ciencia de datos, destacando en automatización, análisis de datos y desarrollo de aplicaciones.

React

React es una biblioteca de JavaScript para construir interfaces de usuario dinámicas y modernas, optimizando el rendimiento y la reutilización de componentes.

Git

Control de versiones permite gestionar cambios en el código y facilitar la colaboración entre desarrolladores, mejorando la eficiencia en proyectos.
